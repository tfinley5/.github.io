Project Requirements and Design


Project Overview

  1. The webpage that I am developing will allow users to learn about web application development and apply the knowledge the learned
  2. The intended users of the site are anyone that would like to learn the basics of web application development
  3. The content on the page covers HTML, CSS, and JavaScript. There will be quizzes on the page so that the users can test the skills that they learned

Page Design

  1. Name: Home
  2. Purpose: Display informarion about the page as a whole and welcomes the user
  3. Users: Customers first impression of site
  4. Content: Welcome message and website overview
  5. Users Entering Data? No
  6. Any data valiations? No
  7. Buttons: None
  8. Actions: There are tabs to navigate in this page

  1. Name: Learn
  2. Purpose: This page is meant for the user to browse and prepare for the quiz section
  3. Users: Customer
  4. Content: The page contains information about HTML, CSS, and JavaScript
  5. Users Entering Data? No
  6. Any data valiations? No
  7. Buttons: None
  8. Actions: The user clicks through the accordian JQuery Tabs to navigate

  1. Name: Quiz
  2. Purpose: Test the users knowledge about HTML, CSS, JavaScript
  3. Users: Customer
  4. Content: Multiple-Choice Quiz
  5. Users Entering Data? Yes, submitting answers to quiz
  6. Any data valiations? Yes, to decide whether the answer is correct or not
  7. Buttons: Button that generates the quiz start and for quiz answers
  8. Actions: Quiz generates when button is clicked, next questions pops up when answer button is clicked

  1. Name: Upcoming
  2. Purpose: Shows future content the webpage will have
  3. Users: Customers
  4. Content: Images and descriptions of what is next to come
  5. Users Entering Data? No
  6. Any data valiations? No
  7. Buttons: None
  8. Actions: None

Dynamic Functionality

  • The functionality in the webpage comes from the quiz page.
  • Using JavaScript, the user will be able to answer the questions and automatically display the next question. When there are no more questions left, the quiz will display a the final score.
