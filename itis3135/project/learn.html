Learn Some Basics

HTML

HTML

  • HTML is all of the structure and content of the page
  • Elements always start with an opening and closing tag and content in between. The type defined in the tags determine the type of output it produces.
  • Some basic types of elements:
    • p - paragraph
    • h1, h2, h... - header tags
    • !DOCTYPE - defines document and must be included at top of html document
    • >html- defines html document
    • title - gives a title for document
    • body - contains content of document
    • header, main, footer - defined in body tag and contain the top, main content, and bottom of page respectively
    • a - defines hyperlink to link other pages

CSS

CSS

  • CSS contains all of the formatting that goes into the page
  • Can be implemented in HTML using external(link), embedded(style) or inline
  • A CSS rule contains a selector, property and a value
  • Selector - determines type of element to format
  • Property - determines the property of the element to format
    • text-align
    • font-size, font-family
    • color
  • Value - determines how to change specific property

JavaScript

JavaScript
